# Extensão para o Chrome em Java Script

Utilizado ferramentas como:
* javaScript
* HTML
* CSS
* Bootstrap
* SweetAlert2

Extensão desenvolvida para realizar controle e validação de informações, realizando requests e realizando encoded base-64.
